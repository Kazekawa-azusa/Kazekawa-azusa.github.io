# 蛋糕
<div class="vertical-wrapper">

從座位上回過神來，思緒還是有點混亂。  
看起來被昨天的經歷嚇得不清。  
這也是正常的吧？  
能在不到幾秒的時間內看完一本書，任誰都無法想像。  
不過人在消化超出自我認知的事時，難免會產生好奇：  
「到底是怎麼做到的呢？」  
抱持著這樣的疑問，放學時間到來了，同學們陸續離開教室，而我仍然無法消化。  
<br/>
轉眼間，放學後的教室只剩夜中同學和我。  
「嚇到了吧 ~」  
思緒被拉了回來，夜中同學站在我的桌子前方看著我，不等我回話就逕自說道  
「我還可以做很多事噢 ~」  
「像是突然出現在某個地點、突然換了身衣服或是讓限量蛋糕憑空出現都是可以的 ~」  
「戶賀同學有想做的事嗎？」  
「疑？」我還在思考夜中同學說的話。  
夜中同學還是一如既往地天馬行空，不過這些應該是辦不到的吧？  
如果是之前的我，一定會這麼認為。  
但是經過了昨天的事，我不禁開始思考她真的做得到。  
<br/>
「蛋米......」我還沒說完夜中同學就拉著我跑出了教室。  
「起司蛋糕，出發！」  
<br/>
在路上，我邊喘著氣，邊斷斷續續的問到  
「為什麼我們要用跑的？」  
「什麼為什麼？我們要去蛋糕店阿」夜中同學流利的回道。  
我停了下來，數秒之後我的呼吸平順了一點，隨後又被夜中同學拉著跑了起來。  
『看起來是假的呢』我在心裡默默想著。  
<br/>
到了蛋糕店，一坐下，我的雙腿就像是融化了一樣使不上力，所以是夜中同學去點餐，而我負責趴在桌上。  
雖然之前也有因為要趕車而要跑步的時候，但是跑那麼遠還是第一次，我的體能果然還是不好。  
<br/>
在我終於有力氣倒水的時候，夜中同學拿著蛋糕回來了。  
「真幸運，剛好只剩最後兩塊了」  
「欸，真的？」沒想到這個時間點還沒賣光，平常不是一開門就賣完了嗎？ 上次也是週末的時候提早來排隊但也無功而返。  
夜中同學把蛋糕放好後就坐到我的對面  
「快點吃吧！ 我可是很有自信的喔 ~」  
夜中同學邊說著中年大廚會說的話，邊帶著期待的眼神直直盯著我看。  
「為什麼搞得好像是妳做的一樣」我吐槽道，隨後放下了水壺，拿起了叉子。  
「我要開動了」  
蛋糕的一小塊黏在了叉子上面，隨後進到了我的口中。  
明明只有一小口，但是起司的濃郁卻毫不遮掩的擴散了開來，而蛋糕的綿密也恰到好處的融成了一塊又一塊。  
「好吃」我發自內心的稱讚著。  
夜中同學此時的表情流露出了幸福感。  
「是吧 ~就說我很有自信了！」  
我笑了出來，可能是因為夜中同學這個奇怪的梗用了兩次，又或者是因為自己好像被當成了小孩子看待而感到不好意思。  
<br/>
吃完了之後，約定好我們明天還要再去玩（其實是夜中同學單方面決定），我們就各自回家了。  
<br/>
不過今天感覺蠻正常的，果然夜中同學是在開玩笑吧！  
還好昨天有猜到會被約出去，所以先跟補習班請好假了，上次這樣無憂無慮的出去玩不知道是多久以前了。  
「明天再請一次假吧。」  
<br/>
回到家的我還是忘不了那個美味的起司蛋糕。  
「好險，不然明天就頭痛了」  
勉強補完作業的我，躺在床上很快就睡著了。
