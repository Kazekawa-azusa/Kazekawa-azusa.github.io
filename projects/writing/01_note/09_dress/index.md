
# 「洋裝」
她很喜歡櫥窗裡的那件洋裝，  
它的造型、剪裁、裝綴 都深得她的心  
但是她買不起，  
卻也忘不掉。  

於是，她  
多打了一份工、少吃了幾次外食、改變了言行  
讓自己配的上那件洋裝。  

終於，她  
存夠了錢，買下了洋裝，穿著走出了門。  
她發現，  
她時時刻刻都在注意著地上的水坑，留意著碗裡的湯汁，當心著動作的幅度，  

隨後，她  
回到了店裡，將它放入了精美的服飾箱中，回到了家。  

隔天，她  
找不到箱子，衝忙出了門，  
走到了那條街，那服飾店或許是搬了家，櫥窗內部只有空蕩蕩的只有幾桶白色油漆。  

最後，她  
也許多打了一份工、少吃了幾次外食、改變了言行  
但她  
仍然忘不了那件洋裝。
