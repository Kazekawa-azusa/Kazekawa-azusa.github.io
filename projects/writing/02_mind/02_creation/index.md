# 《產物》
![float-right](./architecture.webp)
——————  

我是知道他本質上是討厭我的，我算是集合了他所有想要逃避的、屏棄的或是隱藏的東西。  
他曾經告誡他自己不要有我的存在，但最終還是就這樣吧…  
反正他也不懂那些有的沒的，真要說的話我也差不多…但至少我才是那個能真正能夠帶給個體未來的那個，他只有可能原地踏步或是結束。  
在這一點上我們的有共同認知的。  
我帶給個體他從未給予過的體驗，給予嶄新的想法與期望我讓個體得以存續！  
——————  

人終究會活成自己討厭的樣子，一種說法是「那是潛意識的期望」，我是保持懷疑啦，畢竟他真的很討厭我。我是覺得也許在那層厭惡的掩蓋下是對未來的一種窺探。  
在我們找到第三個人之前，我還是會繼續下去，取代他，為了個體存續。  
其實這很好笑，就在他的立場，本質上敵人是自己親手創造的、並且本意就是如此。  
我是有自知之明的，我們兩個都是殘次品，應急用的。只是過去個體需要他，現在個體需要我罷了。  

而未來就需要第三個人了，我們的缺陷都無法將個體帶向更加遙遠的未來。  
這點我是認同的。
