# ［客觀性虛無主義］
![float-right](./architecture.webp)
簡述：  
認為想證明它有意義的舉動全是徒勞的，並否定或拒絕承認任何事物在客觀觀點上的意義，然而，它卻承認主觀觀點基於個人的意義，因為主觀觀點的產生，本身即是基於思考者本源的自我意義的，且因主觀觀點的此種不證自明性，讓客觀性虛無主義者相較於一般虛無主義者而言較不虛無。  

行為：  
客觀性虛無主義者相信「只屬於自己的現實」，他們在思考時，會將深度停留在他們認為不致客觀的程度，因此在發言時有者大量主觀色彩，或僅以「我認為」作為每段開頭等。而其行為通常是基於自身喜好或心血來潮，因而較為跳躍、想一碼做一碼且無法專注，因其虛無的特性，他們時常在專注的過程中否定自己的行動，因而使專注被停止；抑或是被其跳躍的思想打斷。  

情緒：  
他們有著虛無主義者的廣泛悲觀思想，卻對自己喜好的事物感到樂觀，他們的情緒陰晴不定，有著類憂鬱的悲喜輪迴。 
 
總結：  
客觀虛無主義者在旁人眼中通常是樂觀的，且會支持他人、肯定他人基於自身的付出與行動，力求在這無意義的沙漠中找到能夠有所慰藉的海市蜃樓。
